## 1. 개발환경 구축
### 1-1. 다운로드
- anaconda
- vscode

### 1-2. 아나콘다 세팅
- conda env list (가상환경 목록 보기)
- conda create -n developer python=3.11 (가상환경 생성)
- conda activate developer (가상환경 접속)
- pip list (가상환경의 라이브러리 목록 보기)
- pip install pandas (pandas 라이브러리 설치)
- cls (화면 클리어)

### 1-3. vscode 세팅
1. Extenstions 설치
 - Python 설치
 - Prettier 설치
 - Python Extension Pack 설치
 - Atom MAterial Theme
 - Atom Material Icons
2. Settings
 - [Mouse Wheel Zoom] 켜기
3. Thema 설정
4. 아나콘다 가상환경 주입
- [ctrl] + [Shift] + [P] -> "Python: Select Interpreter" 클릭 후
 "developer" 가상환경 클릭

### 1-4. 명령어 단축키
-[crtl] + [,] : Settings 열기
-[ctrl] + [`] : 터미널 열기
